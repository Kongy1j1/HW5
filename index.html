  • Instructions: This app is created by Mingrui Huang and Vincent Wang.
  • This is a one-key input calculator with input by pressing SPACEBAR on key board
    • We designed a single button interface to control the calculator. The cursor will automatically traverse from the first button to the last button of the calculator on the screen. The only operation for the user is when the cursor traverses to the button which the user wants to select, the user needs to press the “space” button to input it into the text box. To texting our design, we respectively set the speed of the cursor as 0.5 seconds and 1.5 seconds. We designed 5 different equations that the tester needs to input, every time the tester gets the result by the calculator, the tester needs to press “C” to zero the calculator. The 5 equations are: 4+3, 7*8+6, (41+4)/9, 874*422, 456+14/2. For every equation, the tester needs to redo 3 times to get an average time.
  • Mingrui Huang worked mainly on the report of tests and feedbacks, and Vincent mainly worked on the coding and tesing.
